RideGuard
Overview

RideGuard is a smart and safety-focused taxi booking web application designed to provide seamless ride management for customers and drivers. The system enables real-time ride booking, GPS-based tracking, secure online payments, and intelligent safety monitoring through AI-powered driver drowsiness detection.

RideGuard combines transportation efficiency with advanced safety technology, ensuring a reliable and secure ride experience for everyone.

Key Features

Customer Features

Book rides with pickup and drop locations
Real-time driver tracking
Dynamic fare calculation based on distance and time
Secure online payment system
Ride history and digital receipts
Rating and feedback system

Driver Features

Driver registration and document submission
Accept or reject ride requests
Manage availability status
View trip history and earnings
Receive ride notifications

AI-Powered Safety Feature

Real-time driver drowsiness detection
Facial recognition and eye movement analysis using OpenCV
Machine learning model for fatigue classification
Instant visual/audio alerts when drowsiness is detected
Driver safety data logging for monitoring

RideGuard is not just a taxi booking system—it is a safety-enhanced smart transportation platform. By integrating real-time tracking, intelligent fare management, secure transactions, and AI-powered driver fatigue detection, RideGuard aims to redefine safe and reliable urban mobility.
